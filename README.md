# SeminarioAlbum

Album de conquistas (achievements) para classes de seminario.

Repositorio contem:
- `web/`: app React + TypeScript (Vite)
- `docs/`: especificacao do produto
- `prototype/`: prototipo em HTML/CSS/JS

## Rodar localmente

```bash
cd web
npm install
npm run dev
```

Build:

```bash
cd web
npm run build
```
