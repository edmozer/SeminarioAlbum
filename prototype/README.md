# Protótipo do Álbum de Achievements

Este é um protótipo funcional navegável do sistema de achievements para seminários.

## Como Executar

1. Abra o arquivo `prototype/index.html` em qualquer navegador moderno.
2. Não é necessário nenhum servidor backend ou build process.

## Funcionalidades Implementadas

### Navegação por Papéis
Na barra lateral, utilize os botões "Admin", "Dir", "Prof" e "Aluno" para alternar instantaneamente entre as visões de cada usuário.

### Telas Principais
- **Dashboard**: Visão personalizada para cada papel.
- **Minha Classe**: Lista de alunos com progresso (Visão do Professor).
- **Detalhe do Aluno**: Histórico e progresso individual.
- **Álbum do Aluno**: Visualização rica das figurinhas/stickers conquistados.
- **Conceder Achievement**: Interface para o professor selecionar alunos e atribuir novas conquistas.

### Tecnologia
- HTML5, CSS3 e JavaScript Vanilla.
- Sem dependências externas além de fontes do Google.
- Design responsivo e tema visual customizado.
